Sistema supervisório

Seja bem vindo ao Supervisório, sistema implementado para a matéria de Controle 2, ministrada pelo Professor Dr. Orides Morandin Junior no período de 2022/2 na Universidade Federal de São Carlos (UFSCar).

Neste primeiro protótipo, os esforços do grupo de desenvolvimento se voltaram principamente ao:

  • Software majoritário do embarcado
  • Software majoritário do supervisório

Nesta primeira versão do software majoritário do supervisório, teremos um servidor local com dados mockados para simular API's REST. Assim, será possível visualizar um sistema com gerenciamento de AGVs, gerenciamento de usuários e login. Com essa estratégia, torna-se possível prorrogar a construção de um back-end que tem o trabalho de manipular os dados num banco de dados, visto que teremos dados de mentira para entender como seria o funcionamento do programa.

Este software realizará o gerenciamento de AGV`s e exibirá o seu status, através de uma conexão socket com o equipamento.
